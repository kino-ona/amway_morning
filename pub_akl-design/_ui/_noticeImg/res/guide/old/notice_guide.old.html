레이아웃

레이아웃 변경점(media query)

  • - 모바일(Mobile S) :width 320px
  • - 모바일(Mobile M) : 화면 폭(width) 375px
  • - 모바일(640px 보다 1px 작음) : 화면 폭(width) 639px
  • - 태블릿(Tablet) : 화면 폭(width) 768px
  • - 데스크탑(Desktop): 화면 폭(width) 1376px

디스플레이 속성 display

  • - display:block .disBl
  • - display:inline .disIn
  • - display:inline-block .disIb (word-break:keep-all 적용됨)
  • - 숨기기 .hide (display:none 아님)
  • - 프린트 할 때 안보이게 .noPrint
  • - PC block, 모바일 inline .txtToInline
  • - PC inline, 모바일 block .txtToBlock
  • - PC block, 모바일 none .pcOnly
  • - PC none, 모바일 block .moOnly
  • - 모바일 none .pcView
  • - PC inline-block, 모바일 none .pcIb
  • - PC none, 모바일 inline-block .moIb
  • - PC block, 태블릿 이하 none .pc
  • - PC none, 태블릿 이하 block .mob

포지션 position

  • - position:absolute .posAb
  • - position:relative .posRe
  • - position:fixed .posFx
  • - PC에서는 absolute 모바일에서는 static .abToSt
  • - PC에서는 relative 모바일에서는 static .reToSt
