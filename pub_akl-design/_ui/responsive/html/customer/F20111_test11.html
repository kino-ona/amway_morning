Amway Korea 홈페이지
사용 안내

30만원 이상 결제는 어떻게 해야 하나요?

  • 1. 이제 공인인증서 없이도 30만원 이상 신용카드 일반결제가 가능합니다.
    *신용카드 일반결제란? 주문 결제 시 간단한 카드 정보 입력만으로 결제가 가능한 신용카드 결제 서비스입니다.
  • 2. 고객센터 보이는 ARS 를 통해 공인인증서 없이 결제 가능합니다.

크롬 설치 및 팝업 해제는 어떻게 하나요?

Amway Korea 홈페이지 최적의 이용을 위해서는 크롬 사용 및 팝업 차단 해제가 필요합니다.

크롬(Chrome) 설치

크롬(Chrome) 다운로드 하기

팝업 차단 해제
  • 1. 우측 팝업 차단 아이콘을 클릭합니다.
  • 2. 클릭 후 화면에 노출되는 “www.amway.co.kr의 팝업을 항상 허용” 을 클릭합니다.
  • 3. 새로고침 후 다시 시도합니다.

로그인은 어떻게 하나요?

  • 1. 우측 상단의 사람 모양 버튼을 클릭합니다.
  • 2. 아이디(회원번호)와 비밀번호를 입력 후 아래 로그인 버튼을 클릭하여 로그인합니다.

마이페이지는 어떻게 들어가나요?

  • 1. 우측 상단의 사람 모양 버튼을 클릭합니다.
  • 2. 마이페이지 메뉴를 클릭하시면 마이페이지 화면을 확인하실 수 있습니다.

나의 주문내역에서 검색 기능은 어떻게 사용하나요?

  • 1. 우측 상단의 사람 모양 버튼을 클릭합니다.
  • 2. 나의 주문 내역을 클릭합니다.
  • 3. 검색 조건을 선택하여 주문내역을 조회합니다.
    예) 기간선택을 누르시면 캘린더 화면에서 기간을 선택하실 수 있습니다.

계좌관리, 포인트관리 메뉴는 어디에 있나요?

마이페이지>결제관리 메뉴 그룹에서 확인 가능합니다.

제품분류(필터) 기능은 어떻게 사용하나요?

제품 목록 화면에서 제품분류(필터) 기능을 이용할 수 있습니다.

  • 1. 필터의 각 항목을 클릭하면 하단으로 목록이 펼쳐집니다.
  • 2. 목록의 항목을 클릭 시 우측 제품리스트가 즉시 변경됩니다.

스마트 오더 캘린더는 어디서 확인하나요?

쇼핑 전체메뉴에서 우측 정기주문&스마트 오더 목록 메뉴를 클릭합니다.

다른 메뉴로 이동은 어떻게 하나요?

마이페이지 및 비즈니스 화면, 암웨이 소개 화면에서는 메뉴 이동 기능을 사용할 수 있습니다.

화면에 보이는 흰색 메뉴상자를 클릭하면 원하시는 메뉴로 이동이 가능합니다.

LOS MAP, 마감실적조회 보기가 불편합니다. 예전처럼 볼 수는 없을까요?

리더님들의 불편을 줄여드리기 위해 기존 ABN 에서 제공하던 LOS MAP 보기와 마감실적조회 화면을 다시 제공할 예정입니다.
