Amway Korea 홈페이지 이용안내

1. 메뉴잠금 설정 재 기능이 아래와 같이 변경되었습니다.
바로 수정을 원하신다면, 아래 [메뉴잠금 설정 바로가기] 버튼을 클릭해 주세요.

메뉴 잠금 설정 바로가기

메뉴 잠금 설정 방법 안내

1. www.amway.co.kr 사이트에 로그인한 후
미니 대시보드의 “마이페이지” 메뉴를 클릭합니다.
마이페이지 화면에서 상단 메뉴 바를 클릭하여 하위 메뉴를 표시합니다.
이 후 마이페이지 → 회원정보 관리 → 메뉴잠금 설정 순으로 클릭하여 “메뉴잠금 설정“ 화면으로 이동합니다.

보안을 위해 휴대폰 본인확인 또는 아이핀 본인확인을
통해 본인인증을 진행합니다.
본인인증에 성공하면 메뉴잠금 설정 화면이 표시됩니다. 이중 잠금 비밀번호 설정 > 버튼을 클릭 합니다.

잠금 비밀번호로 사용할 숫자 4자리를 입력 후
[확인] 버튼을 클릭 합니다.

잠금 설정할 메뉴를 선택 박스를
체크하여 선택한 후 [저장]버튼을 누릅니다.
입력된 4자리 숫자로 메뉴잠금 비밀번호가 설정되었습니다.

2. 로스맵이 아래와 같이 세 가지 모드로 개편되었습니다.

1) 그리드뷰는 데이터 테이블로써 한눈에 많은 정보를 볼 수 있습니다.
2) 트리맵 형식을 통해 한눈에 레그 현황을 볼 수 있습니다.
3) 트리뷰에서 하위회원의 LOS 확장을 통해 원하는 LOS를 상세히 확인할 수 있습니다.

*보다 자세한 로스맵 사용법은 Amway Korea 홈페이지 FAQ를 참조하시기 바랍니다.
https://www.amway.co.kr/customer/cs/faq
