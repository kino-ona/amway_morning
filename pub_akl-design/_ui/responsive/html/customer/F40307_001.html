  2. 2. 장바구니 확인 : 장바구니에 담긴 제품 중 결제할 제품을 선택한 후[주문하기] 버튼을 클릭합니다.
  3. 3. 결제하기 : 결제페이지에서 배송방법, 배송주소, 결제정보 등을 입력 후 [결제요청] 버튼을 클릭합니다.
  4. 4. 주문확인 : 주문이 완료된 후 [주문내역 보기]를 클릭 후, 나의 주문내역을 확인합니다.
  1. ① 제품명(국문)
  2. ② VPS 코드 / SKU 번호 : 전화 주문 시 사용하는 제품 코드 / 제품 발주 시 사용하는 제품 번호
  3. ③ ABO가격, 권장소비자가격
  4. ④ PV(판매점수치 : Point Value) : 각 제품에 부여된 고유 점수치로, 자격 및 후원수당의 비율을 산정하는 기준 점수 BV(판매가격치 : Business Volume) : 각 제품에 부여된 가격지수(가격의 등락에 따라 조정)로, 후원수당의 금액을 산정하는 기준 점수
  • PV와 BV의 관계: 월간 합산된 PV에 의해 보너스 및 후원수당의 비율(3~21%)이 결정되고, 이 비율이 월간 합산된 BV에 곱해지게 됨으로써 보너스 또는 후원수당이 결정됨
  • PV와 BV는 제품 가격이 아니며, 월간후원수당을 산정하는 기준 점수

온라인 팩스(그룹주문) 주문방법

일반 배달 주문과 중개판매 주문을 하실 수 있습니다. 또한, 2인 이상의 ABO 그룹주문이 가능합니다.
그룹주문은 2인 이상의 다수 ABO가 동시 배달 주문하는 서비스로, 주문 총 결제 액이 6만원 이상인 경우 배달 수수료를 절감할 수 있는 주문 서비스입니다.
이때, 실적(PV/BV)는 각각의 ABO 번호로 적립이 되며, 주문방법은 각 지역 암웨이 비즈니스 센터 또는 www.amway.co.kr> 플러스 쇼핑에서 가능합니다. 단, 배송지는 배달 주문 시 원하시는 한 주소지로만 배달됩니다.

결제방법
  1. ① 단일결제 : 신용카드, 자동이체, 무통장입금, A 포인트 결제가 가능하며, 무통장입금은 입금인, 입금은행, 입금 금액을 정확히 입금하셔야 합니다. (단, 3일 이내 입금하지 않는 경우 주문은 자동 취소됩니다.)
  2. ② 그룹결제(ABO 별 카드결제) : 2인 이상의 ABO 그룹 주문으로 각각 결제 시에는 신용카드로만 결제가 가능합니다.
  3. ③ 암웨이 비즈니스 센터 : 신용카드, 현금 또는 자동이체 결제가 가능하며, 한 ABO 분이 결제하셔야 합니다.

단일결제 시 ABO 추가하여 그룹 주문 및 신용카드로 결제한 경우 반품 시 결제하신 신용카드주는 카드사의 규정에 준하며, 이하 그룹 주문한 ABO는 각각의 등록 계좌로 현금 환불 처리됩니다.

  • 주문종류 :일반 배달 주문(그룹주문 가능), 중개판매 주문
  • 주문방법 :
  • 자동이체 결제 / 신용카드 결제 / 무통장 입금 결제 / A포인트 결제 / Re-cash 결제
  • 결제안내
    1. ① 단일결제 : 신용카드, 자동이체, 무통장입금, A 포인트 결제가 가능하며, 무통장입금은 입금인, 입금은행, 입금 금액을 정확히 입금하셔야 합니다. (단, 3일 이내 입금하지 않는 경우 주문은 자동 취소됩니다.)
    2. ② 그룹결제(ABO 별 카드 결제) : 2인 이상의 ABO 그룹주문으로 각각 결제 시에는 신용카드로만 결제가 가능합니다.
    3. ③ 암웨이 비즈니스 센터 : 신용카드, 현금 또는 자동이체 결제가 가능하며, 한 ABO 분이 결제하셔야 합니다.

설치제품 주문방법

설치 주문이 가능한 제품(이스프링 정수기 전환기/퍼싯 타입 및 정수기 필터)에 한해 주문과 동시에 설치/교체 신청이 가능한 주문입니다.
설치주문은 PC/모바일 www.amway.co.kr, 전화주문을 통하여 주문이 가능합니다.
설치/교체 희망일자는 주문일 기준 5일 이내(공휴일 제외)로 날짜 지정 후 주문 완료 후 담당서비스 기사가 연락하여 고객과 협의 후 확정하여 방문합니다.

www.amway.co.kr eSpring "설치/필터 교체 주문" 서비스 이용방법
  1. Step1 : 주문할 제품 선택 설치/필터 교체주 문서비스는 1개 제품만 선택 가능합니다.
  2. Step2 : 섫치주문 클릭.
  3. Step3 : 설치/교체 주소일자 지정 5일이내로 날짜 지정해야 합니다.
  4. Step4 : 결제
  5. Step5 : 주문완료
  6. Step6 : 확정연락 서비스 기사가 방문시간 확정을 위해 연락드립니다.
유의사항
  • 설치/교체 일자를 연기하실 경우 기술지원센터(1588-0080)로 연락 후 변경 가능합니다.
  • 설치/교체 주소 입력 시 해당 설치 ABC에 재고 없을 시, 설치 주문이 불가하니 참고하시기 바랍니다.
  • 지정일 설치 서비스 "지연 지역"이 있으니 반드시 아래 표를 확인 후, 주문하시기 바랍니다.
