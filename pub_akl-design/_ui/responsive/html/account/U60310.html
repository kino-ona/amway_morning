후원자 정정 신청은 본인확인을 통해 계속 진행할 수 있습니다.

휴대폰 본인확인 고객님 명의로 된 휴대폰을 통해
본인을 확인합니다. 휴대폰 본인확인

본인확인이 완료되었습니다.

아이핀 본인확인 고객님의 아이핀과 패스워드를 통해
본인을 확인합니다. 아이핀 본인확인

본인확인이 완료되었습니다.

  • 후원자 정정은 ABO가입 후 3개월 이내에 1회만 하실 수 있습니다.
  • 그룹 누적실적 100만 PV이내에서 신청 하실 수 있습니다.
  • 정정후원자는 동일 다이아몬드 그룹에서 정정 가능합니다.
  • 후원자 정정 신청 절차는 1. 후원자 정정 신청 요청>2. 후원자 정정 승인 중>3. 후원자 정정 완료로 진행 됩니다.
  • ABO님이 후원자 정정 요청을 하시면, 기존 후원자의 승인을 거쳐야만 후원자가 정정됩니다.
  • 기존 후원자 승인 가능 기간은 신청자ABO 가입시점을 기준으로 3개월 이내에 완료되어야 합니다.
  • 기존 후원자가 후원자 정정 요청을 거절하면 신청자의 후원자 정정 요청은 취소됩니다.
  • 요청 이후 현재 스폰서가 승인 전까지 그룹 실적 100만 PV 초과 시 후원자 정정이 취소됩니다.
  • 후원자 정정 완료 후 익일 Amway 사이트 Losmap에서 확인 가능 합니다.
