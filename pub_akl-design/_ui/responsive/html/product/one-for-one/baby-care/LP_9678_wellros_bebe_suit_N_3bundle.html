FAQ
  1. 1) 생산자 직접 배달이란 무엇인가요?
    • 생산자 직접 배달이란 생산 업체에서 제품을 ABO에게 직접 배송하는 서비스입니다. 한국암웨이의 일반 제품은 주문이 접수되면  부산 물류센터에서 제품이 출고되지만, 생산자 직접 배달 주문 제품은 주문 정보가 생산업체로 전달되어 출고됩니다.
  2. 2) 생산자 직접 배달 제품은 ABO 가격이 6만원 미만인데도 배달 수수료가 면제되나요?
    • 생산자 직접 배달 제품은 배달 수수료가 발생하지 않습니다.
  3. 3) 생산자 직접 배달 제품과 일반 제품을 동시에 주문하면 어떻게 배달되나요?
    • 일반 제품은 한국암웨이 부산 물류센터에서 출고되고, 생산자 직접 배송 제품은 생산 업체에서 출고되기 때문에  배송지에 도착하는 시점이 다를 수 있습니다.
