  • Personal Q 유자격 월이란?

    아래 4가지 조건 중 하나 이상을 만족한 월은 Personal Q 유자격월에 해당합니다.

    1. 조건1 : 개인그룹 Personal Q 실적 1,000만 PV 이상 or

    2. 조건2 : 680만 PV 이상 - 1 레그 21% or

    3. 조건3 : 400만 PV 이상 - 2 레그 21%,21% or

    4. 조건4 : 240만 PV 이상 - 3 레그 이상 21%,21%,21% ...

  • 개인그룹 Personal Q 실적이란?

    21% 유자격 레그의 그룹 실적과 현 유자격 플래티늄 이상의 그룹 실적을 제외한 본인의 개인그룹 PV를 의미합니다.

    [A의 개인그룹 Personal Q 실적]

      • A
        • B : PT - 유자격 플래티늄 이상 그룹의 실적은 포함되지 않습니다.
        • C : 15%
        • D : 15%
        • E : 21% - 해당월 21% 유자격 레그의 그룹 실적은 포함되지 않습니다.
        • F : PT - 유자격 플래티늄 이상 그룹의 실적은 포함되지 않습니다.
        • G : 12%
  • 연간 Q-Leg Month 수란?
    • 회계연도 내 본인이 국내에서 직접 또는 대리로 후원하는 월별 21% 유자격 레그 수의 연간 총합을 의미합니다.
    • 연간 Q-Leg Month 수 : 하단 예의 경우, 국내에서 직접 후원하는 21% 자격 레그 수가 6월 : 1개, 7월 : 2개, 8월 : 3개이므로 연간 Q-Leg Month수는 총 6개입니다.

    월별 21% 유자격 레그

    1. 9월 ~5월: 0개

    2. 6월 : 21% 1개

    3. 7월 : 21%,21% 2개

    4. 8월 : 21%,21%,21% 3개=6개

  • Personal Q 보너스 계산의 예

    (2019 회계연도 핀 자격이 2018 회계연도 대비 유지 또는 상승했다고 가정)

    1. 사례 A

      PersonalQ 12 2018 회계연도 Q x 5 개 → PersonalQ 12 2019 회계연도 Q x 5 개: 연간 Q-Leg Month 유지

      2019 회계연도 마감 기준 Personal Q 유자격 월수가 12개이고, 연간 Q-leg Month 수는 전년 수준을 유지하였으므로 700만원을 받습니다. 기본 보너스700 만원
    2. 사례 B

      PersonalQ 11 2018 회계연도 Q x 0 개 → PersonalQ 12 2019 회계연도 Q x 3 개: 연간 Q-Leg Month 3개 증가

      2019 회계연도 마감 기준 Personal Q 유자격 월수가 12개이고, 연간 Q-leg Month 수는 전년 대비 3개 증가하였으므로, 기본보너스 700만원과 추가 성장 보너스 900만원을 받습니다. 기본 보너스700 만원, 추가성장 보너스700 만원 : 총 1,600만원
    3. 사례 C

      PersonalQ 12 2018 회계연도 Q x 0 개 → PersonalQ 11 2019 회계연도 Q x 3 개: 연간 Q-Leg Month 3개 증가

      연간 Q-leg Month 수는 전년 대비 3개 증가하였으나, 2019 회계연도 마감 기준 Personal Q 유자격 월수가 11개이므로 보너스 대상자가 아닙니다. 미수혜
    4. 사례 D

      PersonalQ 12 2018 회계연도 Q x 5 개 → PersonalQ 12 2019 회계연도 Q x 4 개: 연간 Q-Leg Month 3개 증가

      2019 회계연도 마감 기준 마감 기준 Personal Q 유자격 월수가 12개이나, Personal Q 유자격 월 수가 증가하지 않고 연간 Q-leg Month 수는 전년 대비 1개 감소하였으므로, 수혜대상자에서 제외됩니다. 미수혜
    5. 사례 E

      PersonalQ 11 2018 회계연도 Q x 5 개 → PersonalQ 12 2019 회계연도 Q x 4 개: 연간 Q-Leg Month 1개 감소

      2019 회계연도 마감 기준 Personal Q 유자격 월수가 12개이고, 연간 Q-leg Month 수는 전년 대비 1개 감소하였으나 Personal Q 유자격 월 수가 증가하였기 때문에 700만원을 받습니다. 기본 보너스700 만원
