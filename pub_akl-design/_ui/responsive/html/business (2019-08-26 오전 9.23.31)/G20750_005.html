부가가치세의 과세방법

세무 법률 목록

1. 개별등록자

사업자의 유형에 따라 ABO의 소매업과 인적 또는 물적시설을 갖추고 받는 후원수당을 기준으로 부가가치세를 신고납부 하여야 합니다.

1) 일반과세자

납부(환급)
세액계산법
방법1 매출세액 － 매입세액 － 신용카드매출액세액공제
방법2 { 매출과표 × 세율 10% } － { 매입세액 } － { 신용카드발행금액(공급대가) × 1.3% }
  • 매입세액: 매입처별세금계산서합계표, 신용카드매출전표 등 수령 명세서
  • 신용카드매출전표 등 발행세액공제: 연간 500만원 한도(법인사업자와 직전년도 공급가액이 10억 초과하는 개인사업자는 제외)

2) 간이과세자

납부(환급)
세액계산법
{ 공급대가 × 업종별 부가율 × 세율 10% } － 매입세액 × 업종별 부가율 － { 신용카드발행금액(공급대가) × 1.3% }
  • 공급대가 : ABO가격(부가세포함) X 103%
  • 업종별 부가율 : 소매업 10%
  • 매입세액공제 : 매입처별 세금계산서 합계표를 제출하면 매입세액의 10% 공제

2. 총괄등록자

총괄등록자는 전부 간이과세자에 해당됩니다.

납부(환급)
세액계산법
{ 공급대가 × 업종별 부가율 × 세율 10% } － 매입세액공제 － ( 신용카드매출액 × 1.3% )
  • 공급대가 : ABO가격(부가세포함) X 103%
  • 업종별 부가율 : 소매업 10%
  • 매입세액공제 : 매입처별 세금계산서 합계표를 제출하면 매입세액의 10% 공제
