  2. 2ARS 인증 안내

  3. 3생년월일 입력

    ARS 안내 멘트에 따라 생년월일을 전화기에 입력해주세요.

    • 꼭 전화기에 입력하셔야 합니다.

    ARS전화를 종료한 후, 아래의 전화승인완료 버튼을 눌러주시면 인증이 완료됩니다.

    • 반드시 전화승인완료 버튼을 눌러주세요.

  • 이미 등록된 자동이체계좌가 해약된 계좌인 경우, Amway 사이트에서 변경 및 해지가 불가능합니다. ABC로 방문접수 바랍니다.
  • 본인 확인을 위해 전화인증을 수행할 전화번호를 확인 후 [전화 승인요청] 버튼을 선택하여 주시기 바랍니다.
  • 회원정보에 등록된 휴대폰 번호로만 인증이 가능하며, 전화번호가 변경된 경우 "마이페이지 > 회원정보 관리 > 개인정보 수정"에서 수정해 주시기 바랍니다.
