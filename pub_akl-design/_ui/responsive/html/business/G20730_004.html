에메랄드 Depth 보너스

GIP&NCA 목록

1. 대상자

2018 회계연도 마감기준 유자격 에메랄드 이상
(대상자 기준 : 세일즈 & 마케팅 플랜에 의해 전 세계 공통적으로 적용되는 핀 자격 기준)

2. 수혜조건

  1. 1) 국내 유자격 레그 수 3개 이상
  2. 2) 개인별 성장 기준 대비 유지 또는 증가

3. 보너스 금액

  • 유지 시 : 에메랄드 Depth 그룹 BV의 0.5%
  • 성장 시 : 에메랄드 Depth 그룹 BV의 1.0%

4. 개인별 성장기준

본인의 2018 회계연도 12개월 전체 평균 월별 유자격 레그의 수
(직접 또는 대리로 후원하는 레그에 한하며, 평균값 계산시 소수 버림으로 계산됩니다.)

에메랄드 Depth 보너스 그룹 BV 실적이란?

해당 월 본인의 각 유자격 레그의 두 번째 단계(F2) 이하부터 다운라인의 첫 번째 2018 회계연도 마감기준 유자격 에메랄드의 개인그룹과 그가 직접 또는 대리로 후원하는 유자격 레그의 첫 번째 단계(F1)까지를 포함한 총 BV 실적을 의미합니다.

에메랄드 Depth 보너스 그룹 BV 실적의 계산 범위는 해당 월 하위 유자격 에메랄드의 자격 달성여부에 관계없이 2019 회계연도 전체에 걸쳐 동일하게 유지됨을 유의하시기 바랍니다.

에메랄드 Depth 보너스 최소 보장액

(PV:BV = 1 : 1.58 기준 금액이며 , PV : BV 비율에 따라 가변적임)

본인이 직접 또는 대리로 후원하는 각각의 유자격 레그의 첫 번째 단계인 F1의 실적이 1,000만 PV (1,580만 BV)에 이르지 못하는 경우, 해당 레그에 대해서 에메랄드 Depth 보너스 최소 보장액을 상위 유자격 에메랄드에게 보장해 주어야 하며, 이로 인해 상위로 보장되는 (Pass-up) 금액은 본인의 해당 월 에메랄드 Depth 보너스 금액에서 차감합니다.

상기 예의 경우, 2019 회계연도 8월에는 월간 유자격 레그를 3개 이상 달성했으나, 개인별 성장기준 대비 감소한 경우이므로에메랄드 Depth 보너스의 수혜대상에서 제외됩니다.

에메랄드 Depth 보너스 계산의 예 1 (PV : BV = 1 : 1.58 가정)

A의 에메랄드 Depth 보너스 계산

A,G : 2018회계연도 마감 기준 유자격 에메랄드 이상

A(15,800,000 BV), B(4,000,000 BV), C(15,800,000 BV), D(15,800,000 BV), E(15,800,000 BV), F(2,000,000 BV), G(2,000,000 PV), H(15,800,000 BV), I(15,800,000 BV), J(15,800,000 BV)

A의 F1은 B,C,D이고 A의 F2는 E,F이고 G의 F1은 H,I,J 이다

  • ① A(본인) 및 B, C, D의 볼륨: 상위 에메랄드에게 귀속
  • ② 각각의 유자격 레그의 두 번째 단계(F2) 이하부터 다운라인의 첫 번째 2018 회계연도 마감 기준 유자격 에메랄드의 개인 그룹과 그의 해당 월 유자격 레그의 첫 번째 단계(F1)까지에서 발생한 보너스 :
    • E : 1,580만 BV 곱하기 1.0% 는 158,000원
    • F : 200만 BV 곱하기 1.0% 는 20,000원
    • G : 200만 BV 곱하기 1.0% 는 20,000원
    • H : 1,580만 BV 곱하기 1.0% 는 158,000원
    • I : 1,580만 BV 곱하기 1.0% 는 158,000원
    • J : 1,580만 BV 곱하기 1.0% 는 158,000원
    합계 : 672,000원
  • ③ Pass-up된 금액의 차감

    A의 해당 월 유자격 레그의 F1인 B의 실적이 1,580만 BV 미만이므로 상위 에메랄드에게 최소 보장액 158,000원을 보장해주기 위해 Pass-up 되는 금액 118,000원 발생 (최소 보장액 158,000원 – 400만BV X 1.0%)

    해당 예시는 유자격 레그 수가 전월 대비 성장하는 경우를 가정

에메랄드 Depth 보너스 계산의 예 2 (PV : BV = 1 : 1.58 가정)

A의 에메랄드 Depth 보너스 계산

A,E,G : 2018회계연도 마감 기준 유자격 에메랄드 이상

A(15,800,000 BV), B(15,800,000 BV), C(15,800,000 BV), D(15,800,000 BV), E(2,000,000 BV), F(15,800,000 BV), G(4,000,000 PV), H(15,800,000 BV), I(15,800,000 BV), J(15,800,000 BV)

A의 F1은 B,C,D이고 E의 F1은 F,G이다

  • ① A(본인) 및 B, C, D의 볼륨: 상위 에메랄드에게 귀속
  • ② 각각의 유자격 레그의 두 번째 단계(F2) 이하부터 다운라인의 첫 번째 2018 회계연도 마감 기준 유자격 에메랄드의 개인 그룹과 그의 해당 월 유자격 레그의 첫 번째 단계(F1)까지에서 발생한 보너스 :
    • E : 200만 BV 곱하기 1.0% 는 20,000원
    • F : 1,580만 BV 곱하기 1.0% 는 158,000원
    • G : 400만 BV 곱하기 1.0% 는 40,000원
    합계 : 218,000원
  • ③ Pass-up 된 금액의 가산

    A의 하위 2018 회계연도 마감 기준 유자격 에메랄드(E)의 해당 월 유자격 레그의 F1인 G의 실적이 1,580만 BV 미만이므로 이에 대해 최소 보장액 158,000원을 보장받게 되며 이때 Pass-up 되는 금액 118,000원 발생 (최소 보장액 158,000원 – 400만 BV X 1.0%)

    해당 예시는 유자격 레그 수가 전월 대비 성장하는 경우를 가정
