  • 매월 초에는 마감작업으로 인하여 Personal Q 실적 값이 변동될 수 있습니다.
Personal Q 실적이란?
코어 보너스에서 사용하는 개인그룹 루비 PV와 동일합니다.
Personal Q 유자격 월이란?
아래 4가지 조건 중 하나 이상을 만족한 월은 Personal Q 유 자격 월에 해당합니다.
  1. 1. Personal Q 실적이 10,000,000 이상인 경우
  2. 2. Personal Q 실적이 6,800,000 이상 + 1개의 직접 또는 대리로 후원하는 유 자격 레그가 있는 경우
  3. 3. Personal Q 실적이 4,000,000 이상 + 2개의 직접 또는 대리로 후원하는 유 자격 레그가 있는 경우
  4. 4. Personal Q 실적이 2,400,000 이상 + 3개 이상의 직접 또는 대리로 후원하는 유 자격 레그가 있는 경우
