margin + Left{Right/Top/Bottom} + XXL{XL/X/L/ML/M/SM/S} = marLXXL
(최대 크기 PC ~ 최소 크기 모바일, 포토샵 = 모바일 X 2)

padding+ Left{Right/Top/Bottom} + XXL{XL/X/L/ML/M/SM/S} = padLXXL
(최대 크기 PC ~ 최소 크기 모바일, 포토샵 = 모바일 X 2)

- XXL, XL 은 비주얼에서 높이 맞출 때나 많은 간격이 필요할 때(PC/모바일 아주 많은 차이, 자주 사용하지 않음)

- X 는 넓은 폭(PC/모바일 큰 차이, 각 섹션 구분할 때)

- L, ML 은 중간 폭(PC/모바일 중간,약간 차이)

- M, SM, S 좁은 폭(PC/모바일 차이 없음)
