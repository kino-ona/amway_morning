크롬에서 모바일 화면으로 확인하는 방법

  1. 1. 크롬에서 코딩 파일을 불러온다.
  2. 2. 크롬에서 [F12]를 눌러 Developer Tools를 연다.
  3. 3. 열렸으면 [Ctrl + Shift + M]을 누른다.
  4. 4. 모바일 모드의 창 크기를 조절하거나 상단의 Responsive 부분을 조작하여 다양한 크기의 모바일 모드를 테스트한다.
  5. 5. 모바일 모드에서 빠져 나가려면 [F12]를 다시 누른다.

플롯 Float

(모바일에서는 꽉 차서 한 줄로 떨어집니다)

기본은 float이 2개 (50%/50%)   .floR   .floL

<!-- 2개 일 경우 -->
<div class="widM clearfix"> <!-- float 상위에 .clearfix 선언 -->
  <p class="floL">content 1</p>
  <p class="floR">content 2</p>
</div>

float이 3개 이상일 경우 inline style로 퍼센트를 계산하여 추가

<!-- 3개 이상 일 경우 -->
<div class="widM clearfix"> <!-- float 상위에 .clearfix 선언 -->

  <p class="floL" style="width:33%">content 2</p>
