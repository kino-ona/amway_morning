포토샵 -> 모바일 코딩 시 유의사항

포토샵은 모바일 2배수로 되어있기 때문에 포토샵 -> 코딩 시 길이 단위를 1/2 로 해서 작업해야함

ex) 포토샵 2px 두께의 #000 테두리 -> 코딩 CSS에서 border:1px solid #000

ex) 포토샵 반지름 20px의 둥근 모서리 -> 코딩 CSS에서 border-radius:10px

간격(margin/padding), 컨텐츠 폭(widL 등) , 폰트 크기(fntH1 등)는 무조건 2배수는 아니고 가이드를 따라간다.

  • * 암웨이 퀸 체험 클래스는 참석자 요리 실습으로 진행되는 교육 프로그램 입니다.(요리 시연 + 실습), (그룹으로 예약하고 실습 참여 인원은 10명)
