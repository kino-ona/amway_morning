간편가입

후원자 ABO 번호가 자동 입력되는 ABO 간편가입!

간편가입 초대링크를 SMS/카카오톡으로 전송하면
후원자 ABO번호를 직접 입력하지 않아도 간편가입 OK!

초대하실 분의 휴대폰 번호 입력 후, 간편가입 초대링크를 만들어 바로 공유해 주세요!

발송 방법
  • 발급된 URL은 사전에 동의받으신 분께만 전송해야 하며, 위 입력하신 동일한 휴대폰으로 전송해야합니다.
  • 요청하지 않으신 분에게 전송할 경우, 정보통신망법 위반으로 3천만원 이하의 벌금이 부과될 수 있습니다.
