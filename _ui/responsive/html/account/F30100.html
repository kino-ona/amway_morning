엣모스피어 드라이브 제품은 구매 후 30일 이내에 제품 등록 하시면, 제품 등록 시점으로 부터 품질 보증기간이 적용됩니다. 단, 미등록 시 제품 구매일로부터 품질 보증이 적용됩니다.

품질보증이 적용되지 않는 경우
  • 사용자 설명서에 나와 있는 사용지침을 따르지 않거나 다른 용도로 사용 한 경우
  • 남용, 사고, 물리적 손상, 부적절한 설치나 사용, 개조 또는 정비 때문에 손상된 경우
  • 제조업체에서 권장하지 않은 부품이나 액세서리의 사용 때문에 생긴 결함이나 성능 저하는 책임지지 않습니다.
  • 암웨이 제공 설치키트가 아닌 임의적 개조 및 변경(정수기 경우 냉온 정수기 연결, 냉장고 연결 등)
  • 가정용 이외의 용도로 사용한 경우
  • 시리얼 번호를 훼손하여 구입일자가 확인되지 않은 경우
  • 사용자 설명서 미숙지로 제품 고장이 아님에도 서비스를 요청하는 경우
  • 소비자 과실에 의한 고장(충격, 임의분해, 훼손, 부주의 등)
  • 천재지변(낙뢰,화재,염해,수해 등)에 의한 고장
  • 외국에서 판매된 제품
  • 정상 구매 경로가 아닌 경우 (쇼핑몰 사이트 등)
