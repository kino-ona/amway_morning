설치주문 (필터 교체 서비스 주문)

설치주문을 신청하셨습니다. 아래 설치주문 일자를 지정해 주세요.

설치일 지정은 주문일 기준 2~6일 이내(공휴일 제외)입니다. 설치 지정일 이내 미 설치 제품은 주문자에게 자동 배송되오니 유의하시기 바랍니다.

주문일 2018-02-07(수)
  • 정수기 필터의 경우, 교체 서비스 비용은 8,800원(부가세 포함)이며 서비스 당일에 기사에게 결제해 주시면 됩니다.
  • 설치할 배송지 정보는 아래 '배송지 정보'란에 작성해 주세요.
  • 설치서비스 업체에게 설치를 목적으로 한 아래의 개인정보가 제공됨을 알려드립니다.
  • 재고부족으로 설치주문이 불가 할 경우 배달 또는 ABC픽업이 가능합니다.
  • 정수기 본체의 경우, 배달 또는 ABC픽업으로 받으신 경우 3개월 이내 무료로 설치 서비스를 받으실 수 있습니다.
  • 일부 지역의 경우, 보다 원활한 서비스 품질을 위해 특정 요일에 매주 방문 드리고 있습니다. 특정 요일 방문지역은 아래 버튼을 눌러 확인해 주세요.
