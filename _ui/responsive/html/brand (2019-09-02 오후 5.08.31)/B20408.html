전기를 절약하는 정수 시스템

이스프링은 물이 흐를 때만 전원이 켜지는 인스턴트 ON 시스템을 개발하여
불필요한 대기 전력을 줄이고 정수할 때만 효율적으로 전기를 사용합니다.
불필요한 부가기능을 없애고 좋은 물 하나에만 집중하는 기술로 가정의 경제는 물론
지구 환경까지 생각합니다.

구매/유지비

83.28 (1L당 평균 비용* : 약 팔십삼원)

테이블 제목 : 초기 구매 비용, 렌탈료, 5년간 총 비용,1l당 평균 비용
초기 구매 비용 ₩ 1,050,000 (이스프링 퍼싯타입 기준)
렌탈료 없음
필터 구매료 5년간 4회 필터 교체
(₩ 258,000 x 4개 = ₩ 1,032,000)
5년간 총 비용 ₩ 2.082.000
1l당 평균 비용* ₩ 2,082.000 (초기구매비용 + 필터구매료) / (5,000L x 5년) = ₩ 83.28

1년에 5,000리터 사용 기준 5년 사용 시, 정격 유량 3.4L / 분 기준. 1일 사용 시간은 약 4분으로 전기 요금은 제외함.

전기요금

테이블 제목 : 전기요금
전기요금** 하루 1시간 작동 시 (소비전력 60W, 대기전력 2W)
한달 전기요금 ₩ 640 X 12개월 = ₩ 7,680
1년 전기요금 ₩ 7,680 X 5년 = ₩ 38,400

한달 전기요금은 이스프링 정수기 사용에 따라 추가되는 금액이며 1개월 기본 전기 사용량 273KWh
(지식경제부 2013년 1월 21일 발표 ‘2011년 가구당 월평균 전기사용량’)을 기준으로 산출 (한전 사이버지점 홈페이지 참조)
사용량에 따라 차등 적용되는 누진세로 인하여 실제 청구되는 전기요금과 차이가 있을 수 있습니다. (전기요금 기준 : 2014년 6월)
