까다롭게 평가하는 세계적인 인증 기구 에코서트와 프랑스 인증 기구 코스메비오의
인증 마크를 획득한 Amway Korea의 유기농 화장품 브랜드* 입니다.

*보테니컬 립밤의 경우 깔리떼 프랑스(Qualite-France)로
인증 전환되었으므로 상기문구에서 예외 적용됩니다.

Verganic soin thermal 베르가닉(Verganic)이란?

프랑스어 베르(Vert)와 오가닉(Organic)의 합성어 · 베르(Vert) : 그린(Green)을 의미
· 오가닉(Organic) : 유기농 제품을 의미

베르가닉의 모든 제품은

최소 95% 이상 자연에서 추출한
성분으로 만듭니다.

최소 10% 이상 철저하게 인증받은
유기농 성분을 담습니다.

0% 인공색소, 인공향, 파라벤류를
사용하지 않습니다.

유기농 인증마크*를 확인하세요!

코스메비오(COSMEBIO) 오직 유기농 화장품만을 위한 까다로운 인증 마크

에코서트(ECOCERT) 유기농 원료가 포함된 제품에 허락하는 이름

* 유기농 화장품은 원료, 패키지, 포장 등 화장품을 제조 생산 및 운송하는 모든 영역에서 유기농 인증을 받은 제품에 한해 마크를 부여합니다.
