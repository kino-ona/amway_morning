설치제품 주문방법

이스프링 정수기(전환기/퍼싯 타입) 및 정수기 필터 설치 주문방법이 다음과 같이 개선되었습니다.

  1. 1. 장바구니에 다른 제품과 함께 담아서 주문/결제 할 수 있어, 배송비를 절약할 수 있습니다.
  2. 2. 설치 서비스 요청은 결제 완료 이후에 설치 희망일 선택 후 설치 서비스 신청하는 방식으로 변경되었습니다.
  3. ※ 설치 서비스 신청 주소는 배송지 주소와 동일해야 합니다.
www.amway.co.kr eSpring "설치/필터 교체 주문" 서비스 이용방법
  1. Step1 : 주문할 제품 선택 - 설치 서비스 제품(이스프링/필터) 주문 시 서비스 신청주소와 배송지 주소가 동일해야 합니다.
  2. Step2 : 주문 결제
  3. Step3 : 설치 희망일 지정 - 설치 희망일은 주문일 기준 3일 이후부터 지정 가능합니다.
  4. Step4 : 설치 서비스 신청 - 설치를 위한 개인정보 제공에 동의 후 신청합니다.
  5. Step5 : 서비스 방문 연락 - 서비스 기사가 방문시간 확정을 위해 연락드립니다.
유의사항
  • 설치/교체 일자를 연기하실 경우 기술지원센터(1588-0080)로 연락 후 변경 가능합니다.
  • 지정일 설치 서비스 "지연 지역"이 있으니 반드시 아래 표를 확인 후, 주문하시기 바랍니다.
