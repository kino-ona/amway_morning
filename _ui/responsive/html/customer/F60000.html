  1. 주문
    1. 단축번호 1자동응답 주문
    2. 단축번호 2주문 상담사

    1. 단축번호 1배송 조회
    2. 단축번호 2주문 조회 및 취소
    3. 단축번호 3실적 조회
    4. 단축번호 4ABC 전화번호 안내
  3. 상담사 연결
    1. 단축번호 1주문 문의
    2. 단축번호 2배송 문의
    3. 단축번호 3사업 문의
    4. 단축번호 4정수기, 공기청정기 문의
    5. 단축번호 5일반 제품 문의
    6. 단축번호 6뉴트리라이트 전문 상담
    7. 단축번호 7미수금 문의
  4. 기술지원센터 상담사 연결
    1. 단축번호 1정수기 설치/철거 상담
    2. 단축번호 2정수기, 공기청정기 A/S 상담
    3. 단축번호 3기타 상담
