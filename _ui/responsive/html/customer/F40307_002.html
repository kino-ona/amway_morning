신용카드는 주문하는 ABO 본인명의 카드로 결제하셔야 합니다.

대금 지불 방법

주문 제품의 대금은 자동이체, 무통장 입금, 신용카드, 현금(ABC:암웨이 비즈니스 센터에 한함), A포인트/Re-cash, 멤버포인트로 결제가 가능하며 결제 방법에 따라 일일 주문 마감시간이 상이합니다.

자동이체 결제인 경우

[자동이체 계좌 온라인 등록]

  1. ① 마이페이지 >회원정보 변경>계좌관리로 이동 후, 본인 확인 후 등록 신청
  2. ② 자동이체 비밀번호는 6자의 숫자만 입력 가능.
  3. ③ 자동이체 계좌는 주사업자(A&B인 경우 A)의 주민등록번호로 개설된 계좌만이 가능합니다.
  4. ④ 등록 후 실시간 자동이체 서비스 사용 가능
  5. ⑤ 국민, 농협, 유안타, 신한, 씨티, 우리, SC제일, 기업, KEB하나은행(구 외환, 구 하나), 우체국 계좌만 등록 가능

[자동이체의 경우]

  1. ① 자동이체 체결이 되어 있는 ABO는 주문 신청하기 전에 통장 잔액을 미리 확인합니다.
  2. ② 자동이체인 경우 송금수수료는 무료입니다.
  3. ③ 자동이체 가능 시간

무통장 입금 결제 인 경우

  1. ① 다음 한국암웨이 지정 4개 계좌 중 선택한 은행에 주문 금액과 ABO 주사업자 성명 또는 약속한 입금자명으로 정확히 기재하고 입금합니다.
  2. ② 주문일로 부터 3일 이내에 입금하셔야 하며, 3일 경과 시 주문은 자동 취소 됩니다.
