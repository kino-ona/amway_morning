앱 접근 권한 안내

Amway Event App을 사용하실 때
다음과 같은 권한이 필요합니다.

필수적 접근 권한
  • 기기 및 앱 기록
선택적 접근 권한
  • 위치 Amway Event App 위치정보 표시 시
  • 저장공간 Amway Event App에서 제공하는 정보 저장 시
  • 주소록 Amway Event App 내용 공유 시
  • 카메라 QR code 사용시, 사진, 동영상, 파일 첨부 시
  • 알림 Push 알림 서비스 이용 시

선택 접근권한은 해당기능을 사용할 때 허용하며 비 허용 시에도 해당 기능 외 서비스 이용이 가능합니다.

접근 권한 변경 방법
핸드폰 설정 > 앱 또는 어플리케이션 관리
