(분당/강서/인천/대전/울산/부산/창원/전주 ABC 및 A STATION)

이용안내
  • 바우처 1장당 음료 한잔 (3,500원 가격) 선택이 가능합니다. (베이커리, 스낵 불가)
  • 3,500원 이상 음료 선택 시, 추가 금액은 별도 결제 가능합니다. (3,500원 미만 음료는 차액 지불 미적용)
  • 바우처는 현금으로 교환 및 잔액 환불이 불가합니다.
  • 음료 선택 후, 파이토 카페 직원에게 바우처를 제시해 주시기 바랍니다.
유의사항
  • 이용기한은 바우처 표기 기한까지이며, 기한 이후 사용 불가합니다.
  • 이용기한 내 바우처 미사용 시, 분실 시 재발급 불가합니다.
  • e-바우처가 포함된 메시지 삭제 시, 재발송 불가합니다.
  • 양도/양수 가능합니다.

후원수당 및 소득과세 관련 유의사항

다이아몬드 컨시어지 마이랩 마이크로바이옴 솔루션 서비스는 법률상 후원수당 및 사업소득에 해당됩니다.

  • 방판법과 소득세법에 따라, 마이랩 마이크로바이옴 테스트 키트 판매가에 기준하여 소득에 포함되며 종합소득세 신고됩니다.
  • 바우처 사용 유무에 관계없이, 지급 시점에 원수령자인 다이아몬드 이상 리더님께 과세됩니다. 이용기한내에 사용하시길 권장드립니다.
  • 바우처 수령 후, 분실, 미사용, 유효기간 만료된 바우처에 대해서도 소득 취소 불가하며, 과세됨을 알려드립니다.
  • 유자격 시 지급받은 혜택은 기한 내에 사용 가능하며, 자격상실 이후에는 혜택이 제공되지 않습니다.
  • 월별 윤리 강령 이슈자는 다이아몬드 컨시어지 서비스 대상자에서 제외됩니다.
