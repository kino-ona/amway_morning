피부 네트워크 강화와 피부 교정 도움 효과

피부 네트워크 강화 프로-엘라스틴 테크놀로지

여성들의 피부 탄력 개선 효과를 위해 1,000여가지가 넘는 성분들을 테스트 후 피부의 견고함과 탄력을 강화시켜주는 프로-엘라스틴 테크놀로지를 적용시켰습니다.
이 테크놀로지는 피부를 탄탄하게 유지시켜 주고 느슨해진 피부 네트워크를 촘촘하고 단단하게 강화시켜 탄력에 도움을 줍니다.

피부 교정에 도움 레드 캐비어

엄선된 자연 유래 성분, 레드 캐비어 추출물이 함유되어 피부 깊숙이 스며들어 피부 교정을 도와 피부의 젊음과 탄력에 도움을 줍니다. 레드 캐비어 추출물은 피부 속 강화를 도와주며, 피부의 견고함과 탄력을 강화시켜 생기 있고 건강한 피부를 되찾게 해줍니다.

F 앰플리파이어 + 퍼스널라이즈드 씨럼 효과

피부 탄력을 개선시켜주고, 탱탱하게 만들어줍니다.

얼굴 피부의 탄력 증가로 전체적으로 뚜렷하게 보이며, 이마는 더욱 더 눈에 띄게 탄력 있어 보입니다.

인체적용시험에서 92%의 여성의 피부 탄력이 개선되었습니다.
(36명 대상 인체적용시험, 8주 결과)

소비자 조사 결과, 80%의 여성들이 더 탄탄한 피부를 경험하였습니다.
(173명 대상 소비자 설문 조사, 8주 결과)

  • 반품 기준 : 시그니처 셀렉트 앰플리파이어는 제품 특성상 믹스 시 전량 사용되므로 사용 후에는 소비자 만족 보증제도가 적용되지 않았습니다.
    그러나, ABO 리더님들의 지속적인 뷰티 비지니스 성장을 지원하고, 보다 나은 고객 만족을 위해 예외적으로 소비자 만족 보증제도에 따른 반품 규정을 적용하고자 합니다.
    소비자 불만족 반품 시에는 효율적인 처리를 위해 믹스된 퍼스널라이즈드 씨럼과 앰플리파이어 패키지를 반납해 주시기 바랍니다. 기타사항은 소비자 만족 보증제도에 따릅니다.
사용방법
  1. 1) 퍼스널라이즈드 씨럼 상단의 뚜껑을 엽니다.
  2. 2) 앰플리파이어 캡을 퍼스널라이즈드 씨럼에 끼운 후 시계 방향(오른쪽)으로 회전이 멈출 때까지 돌립니다.
    캡이 더 이상 돌아가지 않을 때 시계 반대방향(왼쪽)으로 다시 돌리면 농축된 앰플리파이어가 씨럼에 자동 분사됩니다.
  3. 3) 추가되는 피부 고민을 위한 두 번째, 세 번째 앰플리파이어도 동일한 방법으로 반복해서 믹스해줍니다
  4. 4) 펌프를 퍼스널라이즈드 씨럼에 끼운 뒤, 돌려서 단단히 잠가줍니다.
  5. 5) 퍼스널라이즈드 씨럼과 앰플리파이어가 잘 섞이도록 60초간 잘 흔들어줍니다.
  6. 6) 매일 아침·저녁 토너로 정돈된 피부에 2회 펌핑한 후 아래에서 위로 쓸어 올리듯 얼굴 전체에 부드럽게 펴 바릅니다.
    필요할 경우 1회 펌핑을 더해서 특별히 고민되는 부위에 한번 더 펴 바릅니다.
  • 다음 순서에 따라 최대 3개까지의 앰플리파이어를 1개의 퍼스널라이즈드 씨럼에 믹스해서 사용하시길 바랍니다.
    H 앰플리파이어 - B 앰플리파이어 - W 앰플리파이어 - F 앰플리파이어 - S 앰플리파이어
  • 씨럼 상단의 뚜껑을 엽니다.
  • 앰플리파이어 캡을 퍼스널라이즈드 씨럼에 끼운 후 시계 방향(오른쪽)으로 회전이 멈출 때*까지 부드럽게 돌립니다.
    캡이 더 이상 돌아가지 않을 때 시계 반대방향(왼쪽)으로 다시 돌리면 농축된 앰플리파이어가 씨럼에 자동 분사됩니다.
  • 추가되는 피부 고민을 위한 두번째, 세번째 앰플리파이어도 STEP B 방법으로 반복해서 동일하게 믹스해 줍니다.
  • 펌프를 퍼스널라이즈드 씨럼에 끼운 뒤, 돌려서 단단히 잠궈줍니다.
  • 퍼스널라이즈드 씨럼과 앰플리파이어가 잘 섞이도록 60초간 잘 흔들어줍니다.
  • * 오른쪽으로 회전이 멈출 때 까지 돌릴 시, 끝까지 돌리셔도 별도의 소리가 나지 않습니다.
