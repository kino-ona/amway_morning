레이어팝업 function 가이드
호출

alertShow( 'width' , 'height' , '얼럿ID' )
popupShow( 'width' , 'height' , '팝업ID' )
pageopupShow( '팝업ID' )
slidePopupShow( '팝업ID' )


업데이트(20200720 추가)
- 비동기데이터 대응 업데이트 매서드실행

$( '얼럿ID' )[0].update();
$( '팝업ID' )[0].update();


닫기 function ( 외부제어용 )

얼럿 숨기기
단일 : alertClose( 'hide' , '얼럿ID' )

얼럿 삭제
단일 : alertClose( 'remove' , '얼럿ID' )

팝업 숨기기
단일 : popupClose( 'hide' , '팝업ID' )

팝업 삭제
단일 : popupClose( 'remove' , '팝업ID' )

얼럿 (20200720 유형추가)
defaultdiv.alert_wrap
alertShow( 'width' , 'height' , '얼럿ID' )
팝업 (20200720 유형추가)
defaultdiv.popup_wrap
popupShow( 'width' , 'height' , '팝업ID' )
페이지팝업 (20200720 유형추가)
defaultdiv.pagepop_wrap
pagePopupShow( '팝업ID' )
슬라이드팝업 (20200727 반응형개선)
defaultdiv.slidepopup_wrap
slidePopupShow( '팝업ID' )
